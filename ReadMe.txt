В данной лабораторной работе представлены шаблонные классы для сортировки.

А именно: пузырьковой, Бетчера, вставками, слиянием, кучей(пирамидальной), быстрой, Шелла.

Unit Тесты для них на двух типах данных(int и пользовательским типом данных Person).

Функции для создания файлов с двнными для нагрузочных тестов и сами функции для нагрузочных тестов, 
с графической интерпритацией результатов.

Функция для вывода на экран готовых тестовых данных, ввиду долгой работы пузырьковой и вставками сортировок прикрепляю 
файл с уже готовыми тестовыми данными для сортировок(тестовыми данными выступали int).
Для готовых тестовых данных доступна операция маштабирования, для того чтобы увеличить область выделите ее,
для отмены выделения нажмите на колесико мыши, для возвращения к исходной версии графика нажмите на правую кнопку мыши.

Пользовательский интерфейс со следующим функционалом: открывать файл для редактирования, писать в файл,
автоматически создавать тестовые данные(int), выбирать способы сортировки данных(не представлена сортировка
Бетчера ввиду ограничений на количество элементов для сортировки, а именно 2^n), после каждой операции сортировки 
данные в виде количества элементов, способа сортировки, и времени затраченного на сортировку запоминаются,
реализована модель данных(модель - представление), построение графиков на основе проведенных тестов.
Обратите внимание что после кажой сортировки ее результат отображается в текущем тектовом поле
и если вы хотите выполнить операцию сортировки на случайных данных то текущие данные надо удалить(Ctrl + A).
Обратите внимание что при занесении некорректных данных в тестовое поле они обозначаются красным цветом.
допускается запись одного целого чила в одной строке.

В каждом каталоге с исходными текстами лежит CMakeLists.txt
на случай если вы захотите собирать каждый подпроект руками.

потом build текущем каталоге.
